표준 출력을 할수 있는 방법에 대해 알아보자.
파이썬 - vscode - 3 + 5 입력 후 저장 후 실행 시 8이라는 출력이 나오지 않음.
컴퓨터는 연산을 했으나 출력은 하지 않는다. 출력에 대한 입력이 이루어지지 않았기 때문.

출력 명령 : print
print(3+5) 입력 후 저장 실행 하면 8이 출력 됨.

※ 파이썬은 의미 없는 공백을 입력하면 안됨.(오류 발생)

print('3+5') 입력 후 저장 실행하면 3+5 가 출력됨.
print('3'+'5') 는 35가 출력됨.

' ' 안에 있으면 문자로 인식. ' 문자+문자는 결합연산이다.

print('3' + 5) 는 에러 발생 (종료코드 : 1) <- 비정상종료 / 정상종료코드는 0

str = 문자 / int = 정수
