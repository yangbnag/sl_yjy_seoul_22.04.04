※ 프로그래밍을 가장 효율적으로 잘 할수 있는 방법은 영어 공부-독해.

1. 객체의 키값은 수정 불가 삭제후 재등록

2. 스타일 프로퍼티는 인라인 스타일일때 나옴. 

3. classList는 classname의 문자열로 저장 되어 있는 클래스들을 객체로 보관.

* 함수에 대해 잘 이해가 되지 않을때 -> 검색 ex) js classlist method(함수) 

4. 이벤트 조작 방식에서 어트리뷰트 방식은 태그안에 호출문을 작성 
ex -> <button onclikc = foo();>

5. 이벤트 조작 방식에서 프로퍼티 방식은 이벤트 타겟에 이벤트 핸들러를 넣는다.

$b1.onclick = sayHello(); -> sayHello 함수 호출 후 리턴하는 값을 onclick에 저장 하겠다.
$b2.onclick = sayHello; -> sayHello 함수를 저장 하겠다.

*이벤트 핸들러를 미리 만들어놓고 계속 사용할수 있고, 이름 없는 함수로 만들어 일회용으로 사용 할 수도 있다.
 단점 : 하나의 요소에 하나의 이벤트 핸들러만 바인딩 할수 있다.

6. 이벤트 조작 방식에서 addEventListener('click',funcion(){});
 - click이라는 이벤트에 하나 이상의 이벤트 핸들러 적용가능.
 - 단점 : 이름 없는 함수는 제거를 못한다.
